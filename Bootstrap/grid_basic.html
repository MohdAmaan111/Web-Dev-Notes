The grid system is responsive, and the columns will re-arrange automatically depending on the screen size.

Grid Classes

  • .col- (extra small devices - screen width less than 576px)
  • .col-sm- (small devices - screen width equal to or greater than 576px)
  • .col-md- (extra small devices - screen width less than 576px)
  • .col-lg- (extra small devices - screen width less than 576px)
  • .col-xl- (extra small devices - screen width less than 576px)
  • .col-xxl- (extra small devices - screen width less than 576px)

Tip: Each class scales up, so if you want to set the same widths for sm and md, you only need to specify sm.

The following is a basic structure of a Bootstrap 5 grid:


      <!-- Control the column width, and how they should appear on different devices -->
      <div class="row">
        <div class="col-*-*"></div>
        <div class="col-*-*"></div>
        <div class="col-*-*"></div>
      </div>
      <!-- Or let Bootstrap automatically handle the layout -->
      <div class="row">
        <div class="col"></div>
        <div class="col"></div>
        <div class="col"></div>
      </div>

First example: create a row (<div class="row">). Then, add the desired number of columns (tags with appropriate .col-*-* classes).The first star (*) represents the responsiveness: sm, md, lg, xl or xxl, while the second star represents a number, which should add up to 12 for each row.

Second example: instead of adding a number to each col, let bootstrap handle the layout, to create equal width columns: two "col" elements = 50% width to each col, while three cols = 33.33% width to each col. Four cols = 25% width, etc.

Three Equal Columns

The following example shows how to create three equal-width columns, on all devices and screen widths: 


      <div class="row">
        <div class="col">.col</div>
        <div class="col">.col</div>
        <div class="col">.col</div>
      </div>

Responsive Columns

The following example shows how to create four equal-width columns starting at tablets and scaling to extra large desktops. 


      <div class="row">
        <div class="col-sm-3">.col-sm-3</div>
        <div class="col-sm-3">.col-sm-3</div>
        <div class="col-sm-3">.col-sm-3</div>
      </div>
