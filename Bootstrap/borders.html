Borders

Use the border classes to add or remove borders from an element: 


        <div class="border">1</div>
        <div class="border border-0">2</div>
        <div class="border border-top-0">3</div>
        <div class="border border-end-0">4</div>
        <div class="border border-bottom-0">5</div>
        <div class="border border-start-0">6</div>
        <div class="border-top">7</div>
        <div class="border-end">8</div>
        <divn class="border-bottom">9</div>
        <div class="border-start">10</div>

Border Width

Use .border-1 to .border-5 to change the width of the border: 


        <div class="border border-1">1</div>
        <div class="border border-2">2</div>
        <div class="border border-3">3</div>
        <div class="border border-4">4</div>
        <div class="border border-5">5</div>

Border Color

Add a color to the border with any of the contextual border color classes: 


        <div class="border border-primary">1</div>
        <div class="border border-danger">2</div>
        <div class="border border-success">3</div>
        <div class="border border-warning">4</div>
        <div class="border border-info">5</div>
