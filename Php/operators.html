Operators

Operators are used to perform operations on variables and values.

PHP divides the operators in the following groups:

Arithmetic Operators

The PHP arithmetic operators are used with numeric values to perform common arithmetical operations.

Operator Name Example Result
+ Addition $x + $y 10 + 6 = 16
- Subtraction $x - $y 10 - 6 = 4
* Multiplication $x * $y 10 * 6 = 60
/ Division $x / $y 10 / 6 = 1.6666666666667
% Modulus $x % $y 10 % 6 = 4
** Exponentiation $x ** $y 10 ** 3 = 1000

Assignment Operators

The PHP assignment operators are used with numeric values to write a value to a variable.

Assignment Same as... Example Result
x = y x = y $x = 10;
echo $x;		 10
x += y x = x + y $x = 20;
$x += 100;
echo $x; 		120
x -= y x = x - y $x = 50;
$x -= 30;
echo $x; 		20

$x *= 6;
echo $x; 		30
x /= y x = x / y $x = 10;
$x /= 5;
echo $x; 		2
x %= y x = x % y $x = 15;
$x %= 4;
echo $x; 		3

Comparison Operators

The PHP comparison operators are used to compare two values (number or string):

Operator Name Description Example Result
== Equal Returns true if $x is equal to $y $x = 100;
$y = "100";
var_dump($x == $y); 		bool(true)
=== Identical Returns true if $x is equal to $y, and they are of the same type $x = 100;
$y = "100";
var_dump($x === $y); 		bool(false)
!=
<> 		Not equal Returns true if $x is not equal to $y $x = 100;
$y = "100";
var_dump($x <> $y); 		bool(false)
!== Not identical Returns true if $x is not equal to $y, or they are not of the same type $x = 100;
$y = "100";
var_dump($x !== $y); 		bool(true)
> Greater than Returns true if $x is greater than $y $x = 100;
$y = 50;
var_dump($x > $y); 		bool(true)
< Less than Returns true if $x is less than $y $x = 10;
$y = 50;
var_dump($x < $y); 		bool(true)
>= Greater than or equal to Returns true if $x is greater than or equal to $y $x = 50;
$y = 50;
var_dump($x >= $y); 		bool(true)
<= Less than or equal to Returns true if $x is less than or equal to $y $x = 50;
$y = 50;
var_dump($x <= $y); 		bool(true)
<=> Spaceship Returns an integer less than, equal to, or greater than zero, depending on if $x is less than, equal to, or greater than $y. $x = 5;
$y = 10;
echo ($x <=> $y); 		-1

Increment / Decrement Operators

The PHP increment operators are used to increment a variable's value.

The PHP decrement operators are used to decrement a variable's value.

Operator Name Description Example Result
++$x Pre-increment Increments $x by one, then returns $x $x = 10;
echo ++$x; 		11
$x++ Post-increment Returns $x, then increments $x by one $x = 10;
echo $x++; 		10
--$x Pre-decrement Decrements $x by one, then returns $x $x = 10;
echo --$x; 		9
$x-- Post-decrement Returns $x, then decrements $x by one $x = 10;
echo $x--; 		10

Logical Operators

The PHP logical operators are used to combine conditional statements.

Operator Name Description
and And True if both $x and $y are true
or Or True if either $x or $y is true
xor Xor True if either $x or $y is true, but not both
&& And True if both $x and $y are true
|| Or True if either $x or $y is true
! Not True if $x is not true
