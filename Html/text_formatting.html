Text Formatting

HTML contains several elements for defining text with a special meaning.

Bold Text

Anything that appears within <b>...</b> element, is displayed in bold text. 


        <p>The following word uses a <b>bold</b> typeface.</p>

Output

Italic Text

Anything that appears within <i>...</i> element is displayed in Italic Text 


        <p>The following word uses a <i>italicized</i> typeface.</p>

Output
