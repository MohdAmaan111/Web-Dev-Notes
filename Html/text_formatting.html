Text Formatting

HTML contains several elements for defining text with a special meaning.

Quotation Text

When you want to quote a passage from another source, you should put it in between<blockquote>...</blockquote> tags. 


        <p>Here is a quote from WWF's website:</p>
        <blockquote cite="http://www.worldwildlife.org/who/index.html">
        For 60 years, WWF has worked to help people and nature thrive. As the world's leading 
        conservation organization, WWF works in nearly 100 countries. At every level, we 
        collaborate with people around the world to develop and deliver innovative solutions 
        that protect communities, wildlife, and the places in which they live.
        </blockquote>

Output

Short Quotation Text

The <q>...</q> element is used when you want to add a double quote within a sentence. 


        <p>WWF's goal is to: <q>Build a future where people live in harmony with nature.</q></p>

Output

Abbreviation Text

You can abbreviate a text by putting it inside <abbr>...</abbr> tags.

Use the global title attribute to show the description for the abbreviation when you mouse over the element. 


        <p>The <abbr title="World Health Organization">WHO</abbr> was founded in 1948.</p>

Output

Address Text

The <address>...</address> element is used to contain any address. 


        <p>The HTML address element defines contact information of article.</p>
        <address>
          Written by John Doe.<br>
          Visit us at:<br>
          Example.com<br>
          Box 564, Disneyland<br>
          USA
          </address>

Output

Citation Text

The HTML <cite> tag defines the title of a creative work (e.g. a book, a poem, a song, a movie, a painting, etc.).

The text in the <cite> element usually renders in italic.


        <p><cite>The Scream</cite> by Edvard Munch. Painted in 1893.</p>

Output

Directional Text

The <bdo>...</bdo> element stands for Bi-Directional Override and it is used to override the current text direction. 


        <p><bdo dir="ltr">This text will be written from left to right</bdo></p>
        <p><bdo dir="rtl">This text will be written from right to left</bdo></p>

Output
