Heading Tag

HTML headings are titles or subtitles that you want to display on a webpage.
HTML headings are defined with the <h1> to <h6> tags.
<h1> defines the most important heading.
<h6> defines the least important heading. 


        <h1>Heading 1</h1>
        <h2>Heading 2</h2>
        <h3>Heading 3</h3>
        <h4>Heading 4</h4>
        <h5>Heading 5</h5>
        <h6>Heading 6</h6>

Paragraph Tag

A paragraph always starts on a new line, and is usually a block of text.
The HTML <p> element defines a paragraph. 


          <p>Here is a first paragraph of text.</p>
          <p>Here is a second paragraph of text.</p>
          <p>Here is a third paragraph of text.</p>

Output

Horizontal Rules Tag

The HTML <hr> element is displayed as a horizontal rule.
The <hr> element is used to separate content in an HTML page. 


        <h1>This is heading 1</h1>
        <p>This is some text.</p>
        <hr>
        <h2>This is heading 2</h2>
        <p>This is some other text.</p>
        <hr>

Output

Line Breaks Tag

The HTML <br> element defines a line break.
Use <br> if you want a new line without starting a new paragraph. 


        <p>This is<br>a paragraph<br>with line breaks.</p>

Output

Center Tag

You can use <center> tag to put any content in the center of the page or any table cell. 


        <p>This text is not in the center.

        <center>
        <p>This text is in the center.

        </center>

Output

Preformatted Tag

The HTML <pre> element defines preformatted text.
The text inside a <pre> element is displayed in a fixed-width font, and it preserves both spaces and line breaks. 


        <pre>
          My Bonnie lies over the ocean.
        
          My Bonnie lies over the sea.
        
          My Bonnie lies over the ocean.
        
          Oh, bring back my Bonnie to me.
          </pre>

Output
